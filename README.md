
# Bloom Mood Tracker 🌸
A responsive, interactive web application designed to help users track their emotional well-being and daily activities.

## Features
* **Mood Logging:** Select from 5 distinct mood states with custom icons.
* **Activity Tracking:** Log daily habits like exercise, reading, or socializing[cite: 1].
* **Data Visualization:** Real-time mood trends visualized using **Chart.js**[cite: 1].
* **Local Persistence:** Data is stored securely in the browser's `localStorage`[cite: 1].
* **Data Portability:** Features for exporting and importing your mood history[cite: 1].

## Technologies Used
* HTML5 / CSS3 (Custom Variables & Flexbox)[cite: 1]
* Vanilla JavaScript (ES6+)[cite: 1]
* Chart.js Library[cite: 1]
* FontAwesome Icons[cite: 1]
